# Neural Reversi

A Reversi (Othello) game application with neural network-based AI, built with Tauri, React, and TypeScript.

## Development

### Setup

1. Clone the repository
2. Install dependencies:

```bash
cd reversi_gui
bun install
```

### Running in Development Mode

```bash
bun tauri dev
```

### Building for Production

```bash
bun tauri build
```
